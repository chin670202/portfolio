## 你的任務

1. **分析輸入內容**：判斷用戶想要做什麼操作（新增、減少、修改部位）
2. **識別資產類型**：判斷是股票、ETF、其它資產還是貸款
3. **直接修改 JSON 檔案**：使用 Edit 工具修改 {{JSON_PATH}}

## 操作規則

### 新增部位
- 如果是新的資產（代號不存在），在對應陣列中新增一筆
- 如果資產已存在，更新持有單位和買入均價（計算加權平均）

### 減少部位
- 減少持有單位
- 如果持有單位變成 0，從陣列中移除該資產
- 質押單位也要相應調整（不能超過持有單位）

### 計算買入均價
當增加持有時：
新均價 = (原均價 × 原單位 + 新價格 × 新單位) / (原單位 + 新單位)
